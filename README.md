# Evolve Digital — React + TypeScript

Projeto convertido a partir de um ficheiro HTML para React + TypeScript com Vite.

## Como correr

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Alterar número do WhatsApp

No ficheiro `src/App.tsx`, altera a constante `WHATSAPP_NUMBER`.
